BolashakBot Widget

Умный помощник для студентов вашего университета

Попробуйте прямо сейчас! Виджет уже активен в правом нижнем углу страницы.

Интеграция на ваш сайт

Добавьте всего одну строку кода перед закрывающим тегом </body>:


<script>
    window.BOLASHAK_API_URL = 'https://{{ request.host }}';
</script>
<script src="https://{{ request.host }}/static/js/bolashak-widget.js"></script>
Настройка: Замените https://{{ request.host }} на адрес вашего развернутого BolashakBot.

Дополнительные настройки

Вы можете настроить виджет под дизайн вашего сайта:

<script>
    // Настройка виджета перед загрузкой
    window.BOLASHAK_API_URL = 'https://{{ request.host }}';
    window.BOLASHAK_WIDGET_CONFIG = {
        position: 'bottom-right', // bottom-right, bottom-left
        language: 'ru',           // ru, kz
        theme: 'dark',           // dark, light
        title: 'Помощник Болашак',
        welcomeMessage: 'Здравствуйте! Чем могу помочь?'
    };
</script>
<script src="https://{{ request.host }}/static/js/bolashak-widget.js"></script>
